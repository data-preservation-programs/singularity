# 월렛 관리

{% code fullWidth="true" %}
```
이름:
   singularity wallet - 월렛 관리

사용법:
   singularity wallet command [command options] [arguments...]

명령어:
   import      내보낸 개인 키에서 월렛 가져오기
   list        모든 가져온 월렛 나열
   add-remote  원격 월렛 추가
   remove      월렛 제거
   help, h     명령어 목록 또는 특정 명령어 도움말 표시

옵션:
   --help, -h  도움말 표시
```
