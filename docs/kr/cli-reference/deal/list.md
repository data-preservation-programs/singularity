# 모든 거래 목록 보기

{% code fullWidth="true" %}
```
이름:
   singularity deal list - 모든 거래 목록 표시

사용법:
   singularity deal list [옵션] [인수...]

옵션:
   --dataset value [ --dataset value ]    데이터셋 이름으로 거래 필터링하기
   --schedule value [ --schedule value ]  일정으로 거래 필터링하기
   --provider value [ --provider value ]  공급자로 거래 필터링하기
   --state value [ --state value ]        상태로 거래 필터링하기: 제안됨, 게시됨, 활성, 만료, 제안 만료, 축소됨
   --help, -h                             도움말 보기
```
