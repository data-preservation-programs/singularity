# 딜 일정 예약

{% code fullWidth="true" %}
```
이름(NAME):
   singularity deal schedule - 딜 일정 예약

사용법(USAGE):
   singularity deal schedule command [command options] [arguments...]

커맨드(COMMANDS):
   create   저장 공급자에 딜을 보내기 위한 일정을 생성합니다
   list     딜 생성 일정 목록을 표시합니다
   pause    특정 일정을 일시적으로 중단합니다
   resume   특정 일정을 재개합니다
   help, h  명령어 목록 또는 특정 명령어에 대한 도움말을 표시합니다

옵션(OPTIONS):
   --help, -h  도움말을 표시합니다
```
