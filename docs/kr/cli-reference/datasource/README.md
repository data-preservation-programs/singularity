# 데이터 소스 관리

{% code fullWidth="true" %}
```
이름:
   singularity datasource - 데이터 소스 관리

사용법:
   singularity datasource command [command options] [arguments...]

COMMANDS:
   add      데이터 세트에 새 데이터 소스 추가
   list     모든 소스 나열
   status   데이터 소스의 데이터 준비 요약 가져오기
   remove   데이터 소스 제거
   check    데이터 소스를 확인하여 목록으로 나열합니다. 준비된 항목을 나열하는 것이 아니라 데이터 소스의 연결을 확인하는 것입니다.
   update   소스의 구성 옵션 업데이트
   rescan   데이터 소스 다시 스캔
   daggen   데이터 소스의 전체 폴더 구조를 나타내는 DAG 생성 및 내보내기
   inspect  데이터 소스의 준비 상태 가져오기
   help, h  명령 목록 또는 한 명령에 대한 도움말 표시

OPTIONS:
   --help, -h  도움말 표시
```
