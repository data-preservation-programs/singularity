# 데이터 소스의 준비 상태 확인

{% code fullWidth="true" %}
```
이름:
   singularity datasource inspect - 데이터 소스의 준비 상태 확인

사용법:
   singularity datasource inspect command [command options] [arguments...]

명령어:
   chunks       데이터 소스의 모든 청크 세부 정보 가져오기
   items        데이터 소스의 모든 아이템 세부 정보 가져오기
   dags         생성된 DAG에 대한 모든 피스 세부 정보 가져오기
   chunkdetail  특정 청크에 대한 세부 정보 가져오기
   itemdetail   특정 아이템에 대한 세부 정보 가져오기
   path         데이터 소스 내 경로에 대한 세부 정보 가져오기
   help, h      명령어 목록 표시 또는 특정 명령어에 대한 도움말 표시

옵션:
   --help, -h  도움말 표시
```
