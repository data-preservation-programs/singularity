# 데이터 세트에 새 데이터 소스 추가

{% code fullWidth="true" %}
```
이름:
   singularity datasource add - 데이터 세트에 새 데이터 소스 추가

사용법:
   singularity datasource add command [command options] [arguments...]

명령어:
   acd                Amazon Drive
   azureblob          Microsoft Azure Blob Storage
   b2                 Backblaze B2
   box                Box
   drive              Google Drive
   dropbox            Dropbox
   fichier            1Fichier
   filefabric         Enterprise File Fabric
   ftp                FTP
   gcs                Google Cloud Storage (Google Drive가 아님)
   gphotos            Google Photos
   hdfs               Hadoop 분산 파일 시스템
   hidrive            HiDrive
   http               HTTP
   internetarchive    Internet Archive
   jottacloud         Jottacloud
   koofr              Koofr, Digi Storage 그리고 기타 Koofr 호환 스토리지 공급업체
   local              로컬 디스크
   mailru             Mail.ru Cloud
   mega               Mega
   netstorage         Akamai NetStorage
   onedrive           Microsoft OneDrive
   opendrive          OpenDrive
   oos                Oracle Cloud Infrastructure Object Storage
   pcloud             Pcloud
   premiumizeme       premiumize.me
   putio              Put.io
   qingstor           QingCloud Object Storage
   s3                 AWS, Alibaba, Ceph, China Mobile, Cloudflare, ArvanCloud, DigitalOcean, Dreamhost, Huawei OBS, IBM COS, IDrive e2, IONOS Cloud, Liara, Lyve Cloud, Minio, Netease, RackCorp, Scaleway, SeaweedFS, StackPath, Storj, Tencent COS, Qiniu, Wasabi를 포함한 Amazon S3 호환 스토리지 공급업체
   seafile            seafile
   sftp               SSH/SFTP
   sharefile          Citrix Sharefile
   sia                Sia 분산 클라우드
   smb                SMB / CIFS
   storj, tardigrade  Storj 분산 클라우드 스토리지
   sugarsync          Sugarsync
   swift              OpenStack Swift (Rackspace Cloud Files, Memset Memstore, OVH)
   uptobox            Uptobox
   webdav             WebDAV
   yandex             Yandex Disk
   zoho               Zoho
   help, h            명령어 목록 보기 또는 특정 명령어에 대한 도움말 보기

옵션:
   --help, -h  도움말 표시
```
