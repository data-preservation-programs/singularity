# 관리자 명령어

{% code fullWidth="true" %}
```
이름:
   singularity admin - 관리자 명령어

사용법:
   singularity admin command [command options] [arguments...]

명령어:
   init     데이터베이스 초기화
   reset    데이터베이스 재설정
   migrate  이전 싱귤래리티 몽고디비에서 데이터 마이그레이션
   help, h  명령어 목록 표시 또는 특정 명령어에 대한 도움말 표시

옵션:
   --help, -h  도움말 표시
```
