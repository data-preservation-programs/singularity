# ストレージプロバイダの契約提案のセルフサービスのためのSpade互換APIを開始する

{% code fullWidth="true" %}
```
NAME:
   singularity run spade-api - ストレージプロバイダの契約提案のセルフサービスのためのSpade互換APIを開始する

使用法:
   singularity run spade-api [コマンドオプション] [引数...]

オプション:
   --bind value  APIサーバのバインドアドレス（デフォルト："127.0.0.1:9091"）
   --help, -h    ヘルプを表示
```
