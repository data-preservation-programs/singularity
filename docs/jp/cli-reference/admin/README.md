# 管理者コマンド

{% code fullWidth="true" %}
```
名前:
   singularity admin - 管理者コマンド

使用法:
   singularity admin command [command options] [arguments...]

コマンド:
   init              データベースを初期化します
   reset             データベースをリセットします
   help, h           コマンドの一覧または特定のコマンドのヘルプを表示します

オプション:
   --help, -h  ヘルプを表示する
```
