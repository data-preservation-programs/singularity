# データソースの準備状態を取得する

{% code fullWidth="true" %}
```
NAME:
   singularity datasource inspect - データソースの準備状態を取得する

使用法:
   singularity datasource inspect command [command options] [arguments...]

COMMANDS:
   chunks       データソースのすべてのチャンクの詳細を取得する
   items        データソースのすべてのアイテムの詳細を取得する
   dags         生成されたダグのすべてのピースの詳細を取得する
   chunkdetail  特定のチャンクの詳細を取得する
   itemdetail   特定のアイテムの詳細を取得する
   path         データソース内のパスの詳細を取得する
   help, h      コマンドのリストを表示するか、特定のコマンドのヘルプを表示する

OPTIONS:
   --help, -h  ヘルプを表示する
```
