# データソースのすべてのアイテムの詳細を取得する

{% code fullWidth="true" %}
```
NAME:
   singularity datasource inspect items - データソースのすべてのアイテムの詳細を取得する

USAGE:
   singularity datasource inspect items [コマンドオプション] <source_id>

DESCRIPTION:
   このコマンドは、データソース内のすべてのアイテムをリストします。これは非常に大きなリストになる場合があります。

OPTIONS:
   --help, -h  ヘルプを表示する
```
