# データソースの管理

{% code fullWidth="true" %}
```
名前:
   singularity datasource - データソースの管理

使用法:
   singularity datasource コマンド [コマンドオプション] [引数...]

コマンド:
   add      データセットに新しいデータソースを追加する
   list     すべてのソースを一覧表示する
   status   データソースのデータ準備の概要を取得する
   remove   データソースを削除する
   check    データソースを確認するためにエントリを一覧表示する。これは、準備されたアイテムではなく、データソースの接続を確認するためのデータソースの直接リストです。
   update   ソースの構成オプションを更新する
   rescan   データソースを再スキャンする
   daggen   データソースのフルフォルダ構造を表すDAGを生成してエクスポートする
   inspect  データソースの準備状況を取得する
   help, h  コマンドの一覧を表示するか、1つのコマンドのヘルプを表示する

オプション:
   --help, -h  ヘルプを表示する
```
