# ウォレット管理

{% code fullWidth="true" %}
```
NAME:
   singularity wallet - ウォレット管理

USAGE:
   singularity wallet command [command options] [arguments...]

COMMANDS:
   import      エクスポートされた秘密鍵からウォレットをインポートする
   list        インポートされたすべてのウォレットをリストする
   add-remote  リモートウォレットを追加する
   remove      ウォレットを削除する
   help, h     コマンドの一覧またはコマンドのヘルプを表示する

OPTIONS:
   --help, -h  ヘルプを表示する
```
