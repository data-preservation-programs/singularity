# 检查数据源通过列出其条目。

{% code fullWidth="true" %}
```
命令名称：
   singularity datasource check - 检查数据源通过列出其条目。这不是为了列出已准备的条目，而是直接列出数据源以验证数据源连接。

用法：
   singularity datasource check [命令选项] <source_id> [sub_path]

描述：
   该命令将列出位于<sub_path>下的数据源条目。如果未提供<sub_path>，则将使用根目录。

选项：
   --help, -h 显示帮助信息
```
