# 列出所有交易

{% code fullWidth="true" %}
```
NAME:
   singularity deal list - 列出所有交易

用法:
   singularity deal list [命令选项] [参数...]

选项:
   --dataset value [ --dataset value ]    根据数据集名称筛选交易
   --schedule value [ --schedule value ]  根据计划筛选交易
   --provider value [ --provider value ]  根据提供者筛选交易
   --state value [ --state value ]        根据状态筛选交易: 提议中，已发布，进行中，已过期，提议已过期，已削减
   --help, -h                             显示帮助信息
```
