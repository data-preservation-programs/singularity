# SP Pool management

{% code fullWidth="true" %}
```
NAME:
   singularity deal sp-pool - SP Pool management

USAGE:
   singularity deal sp-pool command [command options]

COMMANDS:
   create              Create a new SP Pool with default deal parameters
   list                List all SP Pools
   get                 Get an SP Pool by ID
   update              Update an SP Pool's default deal parameters
   remove              Remove an SP Pool
   pause               Pause an SP Pool and all its generated schedules
   resume              Resume a paused SP Pool
   add-provider        Add a storage provider to an SP Pool
   remove-provider     Remove a provider from an SP Pool
   add-preparation     Add a preparation to an SP Pool
   remove-preparation  Remove a preparation from an SP Pool
   help, h             Shows a list of commands or help for one command

OPTIONS:
   --help, -h  show help
```
