# Remove an SP Pool

{% code fullWidth="true" %}
```
NAME:
   singularity deal sp-pool remove - Remove an SP Pool

USAGE:
   singularity deal sp-pool remove [command options] <pool_id>

DESCRIPTION:
   Removes the pool. Generated schedules are preserved but unlinked.

OPTIONS:
   --help, -h  show help
```
