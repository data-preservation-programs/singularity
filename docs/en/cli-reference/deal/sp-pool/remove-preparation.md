# Remove a preparation from an SP Pool

{% code fullWidth="true" %}
```
NAME:
   singularity deal sp-pool remove-preparation - Remove a preparation from an SP Pool

USAGE:
   singularity deal sp-pool remove-preparation [command options] <pool_id> <preparation_id>

DESCRIPTION:
   Associated schedules are paused and unlinked, not deleted.

OPTIONS:
   --help, -h  show help
```
