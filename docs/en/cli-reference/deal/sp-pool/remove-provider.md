# Remove a provider from an SP Pool

{% code fullWidth="true" %}
```
NAME:
   singularity deal sp-pool remove-provider - Remove a provider from an SP Pool

USAGE:
   singularity deal sp-pool remove-provider [command options] <pool_id> <provider_id>

DESCRIPTION:
   Associated schedules are paused and unlinked, not deleted.

OPTIONS:
   --help, -h  show help
```
