# Add a storage provider to an SP Pool

{% code fullWidth="true" %}
```
NAME:
   singularity deal sp-pool add-provider - Add a storage provider to an SP Pool

USAGE:
   singularity deal sp-pool add-provider [command options] <pool_id>

OPTIONS:
   --provider value      Storage Provider ID
   --policy value        Replication policy as JSON, e.g. '{"market": 1, "pdp": 1}'
   --url-template value  Optional per-provider URL template override
   --help, -h            show help
```
