# Run the singularity API

{% code fullWidth="true" %}
```
NAME:
   singularity run api - Run the singularity API

USAGE:
   singularity run api [command options] [arguments...]

OPTIONS:
   --bind value  Bind address for the API server (default: "127.0.0.1:9090")
   --help, -h    show help
```
